# User Management Dashboard

A modern, responsive user management dashboard built with React, TypeScript, Material UI, and Vite. This application provides an elegant interface for managing and viewing user data with advanced search, sorting, and pagination features.

## 📸 Screenshots

![User Management Dashboard](./screenshot.png)

## ✨ Features

### Core Features

- **User Listing**: Display users in a beautiful, sortable table
- **Search**: Real-time search by name or email with debouncing
- **Sorting**: Sort users by name in ascending or descending order
- **Pagination**: Responsive pagination with configurable items per page
- **User Details**: Modal view with complete user information
- **URL State Management**: Search, sort, and page parameters persist in URL

### User Experience

- **Responsive Design**: Optimized for mobile, tablet, and desktop
- **Dark/Light Theme**: Material UI theming support
- **Loading States**: Animated loading indicators
- **Error Handling**: Graceful error display with retry functionality
- **Empty States**: Friendly messages when no users match search
- **Smooth Animations**: Hover effects and transitions

### Technical Features

- **TypeScript**: Full type safety throughout the application
- **React Hooks**: Custom hooks for data fetching and state management
- **Clean Architecture**: Reusable components with separation of concerns
- **Performance**: Memoized computations and optimized re-renders
- **Accessibility**: ARIA labels and semantic HTML

## 🚀 Tech Stack

### Frontend

- **React 18** - UI library
- **TypeScript** - Type safety
- **Material UI 5** - Component library
- **Vite** - Build tool and development server
- **React Router DOM** - Routing and URL management
- **Axios** - HTTP client

### Development Tools

- **ESLint** - Code linting
- **Vite** - Fast development server and builds

## 📦 Installation

### Prerequisites

- Node.js (v16 or higher)
- npm or yarn

### Clone the Repository

```bash
git clone https://github.com/m-ailzadeh/user-management-dashboard.git
cd user-management-dashboard
npm install
npm run dev
```
